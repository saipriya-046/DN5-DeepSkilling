# HOL 12 - Ticket Booking App (Conditional Rendering)

## Objectives
Conditional rendering, element variables, preventing components from rendering.

## Task
Guest users see the flight list (GuestPage). Logged-in users see the booking
page (UserPage). Login/Logout button toggles isLoggedIn state, which
conditionally renders the correct page and button label.

## How to run
npm install
npm start
