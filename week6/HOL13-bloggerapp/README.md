# HOL 13 - Blogger App (Conditional Rendering Styles)

## Objectives
Explore multiple ways to conditionally render in React: if/else, ternary,
short-circuit (&&), and element variables.

## Components
- BookDetails: if/else statement
- BlogDetails: ternary operator
- CourseDetails: && short-circuit + element variable

## How to run
npm install
npm start
