# HOL 11 - Event Examples App

## Objectives
React events, event handlers, synthetic events, this-binding.

## Components
- Counter: Increment/Decrement buttons; Increment also calls sayHello()
- Welcome: button invokes a function with an argument ("welcome")
- ClickMe: synthetic event onPress shows "I was clicked"
- CurrencyConvertor: handleSubmit converts INR to EUR (approx rate 90) on submit

## How to run
npm install
npm start
