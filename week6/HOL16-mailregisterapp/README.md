# HOL 16 - Mail Register App (Form Validation)

## Objectives
React form validation via event handlers and event submit, controlled
components.

## Validation rules
- Name: at least 5 characters
- Email: must contain @ and .
- Password: at least 8 characters

Errors are computed in validate() on submit and shown per-field; a green
success message appears once all fields pass.

## How to run
npm install
npm start
