# Cognizant DN 5.0 Deep Skilling

This repository contains my solutions, hands-on exercises, and learning artifacts for the Cognizant Digital Nurture (DN) 5.0 Deep Skilling Program.

## 📁 Repository Structure

### Week 1 — Data Structures, Algorithms, Design Patterns & SOLID Principles

| Folder | Description |
|--------|-------------|
| `Week1/DataStructures/` | Arrays, LinkedList, Stack, Queue, Dictionary, List, Strings |
| `Week1/Algorithms/` | BinarySearch, LinearSearch, BubbleSort, InsertionSort, SelectionSort, MergeSort, QuickSort, HeapSort, Analysis |
| `Week1/DesignPatterns/` | Singleton, Factory Method, Abstract Factory, Builder, Prototype |
| `Week1/SOLID/` | SRP, OCP, LSP, ISP, DIP demos with C# examples |
| `Week1/NUnitDemo/` | Unit testing with NUnit — Calculator tests |
| `Week1/MoqDemo/` | Mocking with Moq — OrderServices, UserServices, EmailServices |
| `Week1/SQL/` | SQL scripts: DDL, DML, Joins, Subqueries, Views, Stored Procedures, Functions, Triggers, Indexes, Transactions |
